# Live Demo

Below is a fully functional instance of the OSDLabel annotator. You can draw, select, move, and resize annotations on the Deep Zoom Images.

:::tip[Controls]
- **Draw:** Select a tool from the toolbar and drag on the image.
- **Select:** Use the cursor tool (V) to select annotations.
- **Pan/Zoom:** Use scroll to zoom. Hold `Ctrl` (or `Cmd`) to pan while in drawing mode.
:::
