# Installation

Install `osdlabel` and its peer dependencies using your preferred package manager.

:::tip
This library requires `fabric` >= 7.1.0 and `openseadragon` >= 5.0.0.
:::

## Using pnpm (Recommended)

```bash
pnpm add osdlabel fabric openseadragon solid-js
```

## Using npm

```bash
npm install osdlabel fabric openseadragon solid-js
```

## Using yarn

```bash
yarn add osdlabel fabric openseadragon solid-js
```

## Peer Dependencies

The library relies on `solid-js` for its component architecture. Ensure your project is configured with SolidJS support (e.g., using `vite-plugin-solid`).
